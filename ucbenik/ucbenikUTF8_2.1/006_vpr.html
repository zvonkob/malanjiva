Vprašanja z izbiro

  1. S katerim elementom ustvarimo gumb za odpošiljanje obrazca?
    1. input
    2. submit
    3. reset
    4. mailto
  2. S katero lastnostjo povežemo več izbirnih gumbov v skupino?
    1. name
    2. id
    3. value
    4. type
  3. S katerim elementom ustvarimo potrditveno polje?
    1. input
    2. checkbox
    3. radio
    4. reset
  4. Kako pravilno ustvarimo gumb z napisom "Briši", ki bo povrnil vsebino obrazca v začetno stanje?
    1. <input type="reset" value="Briši" />
    2. <input type="reset">Briši</input>
    3. <input type="submit" value="Briši" />
    4. <input type="submit">Briši</input>
  5. Kateri element lahko vstavimo direktno v element select ?
    1. option
    2. input
    3. radio
    4. checkbox
  6. V nekem obrazcu imamo element <input type="checkbox" checked="checked" id="ekstra" /> Kakšen zapis bo element povzročil v odposlanem dokumentu, če uporabnik pred odpošiljanjem obrazca ni ničesar spreminjal?
    1. ekstra=
    2. ekstra=on
    3. ekstra=checked
    4. Nič od naštetega.
  7. Kako vpišemo začetno besedilo urejevalnega polja (element textarea)?
    1. Kot njegovo vsebino.
    2. Prek lastnosti value.
    3. Prek lastnosti name.
    4. Začetne vsebine ne moremo vpisati.
  8. Kako ustvarimo polje za vnos krakega besedila z že vpisanim besedilom "Zdravo!"?
    1. <input type="text" id="pozdrav" value="Zdravo" />
    2. <input type="text" id="pozdrav">"Zdravo"</input>
    3. <input type="text" id="pozdrav" name="Zdravo" />
    4. <input type="text" id="Zdravo" />
  9. Kako ustvarimo seznam v obliki poteznega menija z možnostjo izbire le ene možnosti?
    1. <select size="1" multiple="off">...</select>
    2. <select>...</select>
    3. <select size="1" multiple="multiple">...</select>
    4. <select size="0">...</select>
  10. Kakšnega tipa je element input?
    1. Prazen in blokovski.
    2. Prazen in vrstičen.
    3. Z vsebino in blokovski.
    4. Z vsebino in vrstičen.

Stran dopolnite tako, da bo možno zahtevek za rezervacijo poslati na nek e-poštni naslov. V poslanem dokumentu morajo biti jasno razvidni termini, ki jih je uporabnik rezerviral.

Z znanjem, ki ga imamo, ni možno izdelati dinamične strani, ki bo avtomatično zabeležila rezervacijo in ob naslednjem nalaganju prikazala novo stanje. Za to potrebujemo nek strežniški program, ki ga napišemo v jezikih, kakršna sta PHP ali ASP. Pisanje strežniških programov presega okvire našega učbenika.
